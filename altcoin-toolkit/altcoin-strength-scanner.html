📖 Score Calculation Methodology

×

✅ Hybrid Scoring System

Fast loading with basic CoinMarketCap scores, then on-demand fundamental analysis via Perplexity AI when you click on specific tokens. Best of both worlds: speed + depth.

🎯 Fundamental Score System

Total Score = Sum of 20 Individual Metrics (0-10 each) = Maximum 100 Points

Each fundamental metric is scored 0-10 based on predefined thresholds and peer comparisons

💰 Token Economics & Supply (25 points)

Metrics (5 × 5 points each):

  • Token Unlock Schedule (0-10): Lower unlock percentage = higher score
  • Total Value Locked (0-10): Higher TVL = higher confidence
  • Supply Dynamics (0-10): Deflationary > Stable > Inflationary
  • Market Capitalization (0-10): Larger cap = higher confidence
  • Token Distribution (0-10): Less concentration = better score

🌐 Network Activity & Adoption (25 points)

Metrics (5 × 5 points each):

  • Active Addresses (0-10): More daily/monthly addresses = higher score
  • Transaction Volume (0-10): Higher USD value of transactions
  • Protocol Revenue (0-10): Monthly revenue generation
  • User Count (0-10): Number of active protocol users
  • Fully Diluted Valuation (0-10): Reasonable valuation ranges

🔧 Technical & Development (25 points)

Metrics (5 × 5 points each):

  • Development Activity (0-10): GitHub commits in last 30 days
  • Security Audits (0-10): Number of completed security audits
  • On-chain Activity (0-10): Transaction growth trends
  • Liquidity (0-10): Total liquidity across exchanges
  • Trading Volume (0-10): 24-hour trading volume

👥 Market & Community (25 points)

Metrics (5 × 5 points each):

  • Community Size (0-10): Combined social media followers
  • Partnerships (0-10): Number of major partnerships
  • Ecosystem Flows (0-10): Cross-chain transaction volume
  • P/F Ratio (0-10): Price-to-Fees ratio (lower = better value)
  • MCap/TVL Ratio (0-10): Market Cap to TVL ratio (lower = better)

📊 Data Sources & API Integration

Primary Data Sources (Real-time via Perplexity AI):

  • ✅ Perplexity AI API: Real-time fundamental metrics research
  • ✅ CoinMarketCap API: Price, market cap, volume data
  • ✅ On-chain Analytics: GitHub, DeFiLlama, protocol data
  • ✅ Social Metrics: Twitter, Discord, community data

Scoring Integration:

  • 20 Fundamental Metrics: Each scored 0-10 points
  • Real-time Research: Perplexity AI fetches latest data
  • No Price Speculation: Focus on solid fundamentals only
  • Threshold-based Scoring: Objective evaluation criteria

🎯 Fundamental Score Interpretation

  • 85-100: Elite fundamental strength - institutional grade
  • 70-84: Strong fundamentals - high conviction projects
  • 55-69: Solid fundamentals - good potential
  • 40-54: Mixed fundamentals - moderate risk
  • 25-39: Weak fundamentals - high risk
  • 0-24: Poor fundamentals - extreme risk

Note: Scores are based purely on fundamental analysis, not price performance. A project scoring 85+ demonstrates exceptional fundamentals across token economics, adoption, development, and community metrics.
